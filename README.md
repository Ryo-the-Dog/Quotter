# 概要
本や映画の好きなフレーズを「引用」(quote)して共有するサービスです。

# 開発環境・使用言語
MAMP
<br>
HTML・Sass(SCSS)・Laravel・Vue.js・MySQL

# 制作について
要件定義・デザイン・コーディング・DB設計・デプロイなど全て独力で行いました。
<br>
会員機能、バリデーション、投稿、画像アップロード、いいね機能などのwebサービスに必要な機能を一通り実装しています。
<br>
また、いいね機能はaxiosを使った非同期通信で実装しました。
